# Деплой бота на Railway

## Переменные окружения (обязательные)

| Переменная | Описание |
|---|---|
| `BOT_TOKEN` | Telegram bot token от @BotFather |
| `NOTIFY_CHAT_ID` | ID чата/канала для уведомлений о новых отзывах |

## Переменные окружения (опциональные)

| Переменная | По умолчанию | Описание |
|---|---|---|
| `PODCAST_ID` | `1371411915` | Apple Podcasts ID подкаста |
| `ADMIN_IDS` | `869587` | Telegram user IDs админов (через запятую) |
| `EPISODE_RSS_URL` | transistor.fm feed | RSS-лента для мониторинга эпизодов |

## Автоматический деплой

Push в main → Railway деплоит автоматически.

## Как узнать NOTIFY_CHAT_ID

1. Добавьте бота в канал/чат
2. Отправьте `/set_notify` (только для админов)
3. Бот покажет chat ID — установите его в env vars на Railway
